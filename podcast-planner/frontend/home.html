🚀 Quick Start

  1. Open Agent Pipeline to see the 5-agent workflow
  2. In VS Code, press Ctrl + Shift + I to open Copilot Chat
  3. Set mode to "Agent" at the bottom of the chat
  4. Click each agent card and copy the prompt to Copilot Chat
  5. Run agents 1 → 2 → 3 → 4 → 5 in sequence
  6. Check shared/project_context.md for the complete output
