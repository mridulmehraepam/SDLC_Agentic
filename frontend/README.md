# Frontend - Podcast Episode Planner + Script Assistant

## Run

Since this is a no-build frontend, you can run it directly:

1. Open `index.html` in your browser.
2. Or use VS Code Live Server extension and open the `frontend` folder.

## Features Included

- Episode Planner Dashboard (search + status filter)
- Create/open episode workflow
- Episode Workspace (metadata, guests, outline, script)
- AI Assistant panel (generate outline/script, regenerate, run history)
- Status actions (save draft, save version, mark ready)
- Responsive layout for smaller screens

## Notes

- AI actions are mocked in the browser (`app.js`) to simulate async behavior.
- Data is currently in-memory and resets on page refresh.
